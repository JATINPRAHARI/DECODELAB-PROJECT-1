# PROJECT-1: Rule-Based AI Chatbot 🤖

**DecodeLabs Internship Project 1**

A deterministic Rule-Based AI Chatbot built using pure Python with logic gates and dictionary-based knowledge base. No machine learning — 100% transparent and explainable.

---

## 📋 Project Overview

This project demonstrates:

* Rule-based intent matching
* Input sanitization and preprocessing
* Dictionary-based knowledge base (O(1) lookup)
* Control flow and decision logic
* Clean, well-documented object-oriented code

---

## 🚀 Features

* **Greeting Handling** (hello, hi, good morning, etc.)
* **Project Information** (Project 1 & Project 2)
* **Technical Concepts** (KNN, Iris Dataset, Feature Scaling)
* **Smart Partial Matching** as fallback
* **Exit Commands** (bye, exit, quit)
* **Professional Welcome Screen**
* **Error Handling & Session Summary**

---

## 🛠️ Technologies Used

* Python 3.x
* No external libraries required
